# CALL
CALL stands for:<BR>
**C**ommon<BR>
**A**ssembly<BR>
**L**ike<BR>
**L**anguage<BR>
CALL is an interpred language written in python, in the future we should
implement a bytecode/binary code compiler but for now is just an interpreted
language
